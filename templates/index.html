Jobcentre API

This service acts as a proxy for los.direct.gov.uk. It lets you look up the jobcentre responsible for a given postcode.

Documentation

You can search by postcode: /api?postcode=SW1A2LW

A successful response looks like this and returns a 200 status code:
{ "address": "Chadwick Street, London, United Kingdom, SW1P 2ES", "name": "Westminster Jobcentre Plus", "opening-times": [], "postcode": "SW1P 2ES" }
